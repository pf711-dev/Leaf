这是一篇示例文档，用于预览布局效果。实际应用中，右侧区域会通过 iframe 加载用户导入的 HTML 文档。

切换上方「展开 / 收起」按钮对比两种布局。

容器组件与展示组件

容器组件负责数据获取与状态管理，展示组件只关心 UI 渲染。这种分离让组件更易于复用与测试。

自定义 Hook

将组件逻辑提取到可复用的函数中，是 React Hooks 时代最重要的模式之一。例如 useFetch、useLocalStorage 等。

组合优于继承

通过 props 传递组件、使用 children 进行组合，比深层的继承层级更灵活，也更符合 React 的函数式风格。
