# Leaf

基于 Tauri + Vue 3 的本地 HTML 文档阅读库。

## 技术栈

- Tauri 2（Rust 后端）
- Vue 3 + TypeScript + Vite
- Pinia 状态管理

## 开发

```bash
npm install
npm run tauri dev
```

## 构建

```bash
npm run tauri build
```
